
Django Subcommand
==================
Add sub command to A Django manage command.


Installation
~~~~~~~~~~~~

.. highlight:: python

edit settings.py ::

    INSTALLED_APPS = (
        "subcommand",
    )

Setup
=====

.. highlight:: bash

::

    $ pip install django-subcommand
